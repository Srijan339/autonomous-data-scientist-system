### Problem

Most organizations generate large volumes of data but lack skilled data scientists to analyze it effectively. Data science workflows require extensive manual effort for data cleaning, exploratory analysis, feature engineering, and model selection.

### Goal

Build an **Autonomous Data Scientist System** that automates the end-to-end machine learning pipeline including:

* data profiling
* exploratory analysis
* feature engineering
* model training
* model evaluation
* explainable AI
* insight generation

### Impact

Reduce manual data science effort while enabling  **non-experts to extract insights from data** .


Research Questions

1. Can an automated pipeline replicate the core workflow of a data scientist?
2. How does automated model selection compare to manual model development?
3. Can AI-generated insights improve interpretability of machine learning outputs?
